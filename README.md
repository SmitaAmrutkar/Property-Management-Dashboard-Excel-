# 🏢 Property Management Dashboard

## 📌 Project Overview
The Property Management Dashboard is an Excel-based data analytics project designed to provide insights into property performance, occupancy rates, rental income, and maintenance expenses. The dashboard enables property managers and stakeholders to monitor key metrics and make informed decisions through interactive visualizations and reports.

## 🎯 Project Objectives
- Track property occupancy and vacancy rates.
- Monitor rental income and revenue performance.
- Analyze maintenance and operational expenses.
- Evaluate overall property performance using KPIs.
- Provide actionable insights through interactive dashboards.

## 🛠️ Tools & Technologies
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning & Analysis

## 📊 Dashboard Features
- Occupancy Rate Tracking
- Rental Income Analysis
- Maintenance Cost Monitoring
- Property Performance KPIs
- Interactive Filters and Slicers
- Monthly Revenue Trends
- Expense Breakdown Analysis

## 📈 Key Metrics
- Total Properties
- Occupied Units
- Vacant Units
- Occupancy Rate (%)
- Total Rental Income
- Maintenance Expenses
- Net Revenue

## 🔍 Insights Generated
- Occupancy trends across properties.
- Revenue performance over time.
- High-cost maintenance areas.
- Property-wise performance comparison.
- Opportunities to improve profitability.
